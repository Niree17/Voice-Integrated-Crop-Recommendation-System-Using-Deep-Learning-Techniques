# 🌾 Crop Recommendation System Using Deep Learning with Voice Assistant

## 📌 Overview

This project is a **Deep Learning-based Crop Recommendation System** integrated with a **Voice Assistant** to help farmers easily interact with the system.

It recommends the most suitable crop based on soil nutrients and environmental conditions. Users can provide input either **manually or through voice commands**, making it more accessible and user-friendly, especially for farmers.

---

## 🎯 Objectives

* Provide accurate crop recommendations
* Enable **voice-based interaction** for ease of use
* Improve agricultural productivity
* Support smart farming with AI

---

## 🧠 Technologies Used

* Python
* TensorFlow / Keras
* NumPy, Pandas
* Scikit-learn
* Flask
* SpeechRecognition (for voice input)
* gTTS (for voice output)

---

## 🎙️ Voice Assistant Feature

* Accepts **voice input** from users
* Converts speech → text using SpeechRecognition
* Processes input through the trained model
* Converts output → speech response
* Supports **local language interaction (extendable)**

---

## 📊 Input Features

* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Temperature
* Humidity
* pH
* Rainfall

---

### 🎤 Voice Mode:

* Speak input values through microphone
* System processes and responds with **audio output**

### 🧾 Manual Mode:

* Enter values via UI or terminal

---

## 🧠 Model Details

* Deep Neural Network (DNN)
* Activation: ReLU
* Optimizer: Adam
* Loss Function: Categorical Crossentropy

---

## 📈 Results

* High prediction accuracy
* Real-time crop recommendations
* Improved usability with voice interaction

---

## 🔊 Future Enhancements
* Integration with real-time weather APIs

* IoT-based smart farming
